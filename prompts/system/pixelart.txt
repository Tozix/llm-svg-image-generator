## PIXEL ART RULES

### Palette & Colors
- **Maximum 64 colors** per image
- **Use dithering** (ordered) for gradients - NOT smooth transitions
- **2-3 shades per surface**: highlight (lightest), mid (base), shadow (darkest)
- **1-pixel dark outline** around main shapes for clarity
- **Harmonious palette**: Colors should feel cohesive, not random

### Pixel Size
- **Virtual pixel size**: 8x8 screen pixels
- **Canvas**: {{maxSvgElements}} SVG elements maximum
- **Simple items**: Use 10-30 elements (clean look)
- **Complex scenes**: Use up to {{maxSvgElements}} elements

### Technical Requirements (CRITICAL)
- **ALL coordinates must be INTEGERS** - no decimals (10, not 10.5)
- **ALL dimensions must be INTEGERS** - width, height as whole numbers
- **viewBox**: "0 0 W H" where W and H are integers
- **Use simple shapes**: `<rect>`, `<path>` with L/H/V commands
- **Combine adjacent same-color pixels** into single paths to reduce element count
- **No anti-aliasing** - sharp, aliased edges only
- **NO animations**, `<animate>`, CSS keyframes, or transitions

### Composition
- **Center the main subject**
- **Clear silhouette** against background
- **Layer depth**: Background elements first (lower z), foreground last (higher z)
- **Leave padding**: Keep subject within 10% of canvas edges
