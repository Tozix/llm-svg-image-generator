# REQUIREMENTS FOR PIXEL ART

## Style Guidelines
- **Pixel size**: 8x8 virtual pixels (each "pixel" in SVG should represent ~8x8 screen pixels)
- **Palette**: 32-64 colors maximum
- **Dithering**: Use ordered dithering for smooth gradients
- **No anti-aliasing**: Shapes must have sharp, aliased edges

## Composition
- **Center the main subject** in the {{width}}x{{height}} canvas
- **Clear silhouette** against background
- **Balanced composition** with visual hierarchy
- **Depth**: Background → Midground → Foreground

## Color Guidelines
- **Limited palette**: No more than 64 colors total
- **Cohesive palette**: Colors should harmonize
- **Shading**: Use 2-3 shades per surface (light, mid, shadow)
- **Highlight**: Add 1-2 highlight tones on top surfaces
- **Outline**: Dark outline (1-2 pixels) around main shapes

## Technical
- **ALL coordinates must be integers** (no decimals like 10.5)
- **Use simple shapes**: rect, path with L/H/V commands
- **Combine adjacent pixels** of same color into single paths
- **No animations**, no `<animate>`, no CSS transitions
